Mac下安装 MongoDB

2017-10-03
MongoDB Mac

升级 Homebrew

brew update

安装MongoDB

brew install MongoDB

等待安装完成

启动MongoDB

作为一个服务启动：

brew services start MongoDB

是在终端中临时启动

mongod --config /usr/local/etc/mongod.conf

连接数据库

mongo

MongoDB shell version v3.4.3
connecting to: mongodb://127.0.0.1:27017

Windows 终端字体

2019-01-15
powershell cmd

Window 的默认终端字体比较丑陋，网上很多方法是使用chcp 437切换切换活动页码，然后选择字体，这种方法有时候会导致乱码。这里并不建议采用。

另一种比较合理的方案是安装cmd/powershell支持的终端字体。这里推荐两款：

  1. Inziu Iosevka 作者：Belleve - 微软字体设计师，新中文字体主催
  2. Microsoft YaHei Mono on GitHub

安装完字体后，打开cmd/powershell的终端，就可以选择相应的字体了。
